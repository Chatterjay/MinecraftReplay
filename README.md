# MinecraftReplay
用于存放我的世界的录播replay文件

### 版本分类
|    地图     |   游戏版本   |                        相关录播                        |
|:---------:|:--------:|:--------------------------------------------------:|
|   不折之心    |  1.20.4  | [链接](https://www.bilibili.com/video/BV1D1eqeiE6o/) |


### 食用方法
- 在装有ReplayMod的情况下打开游戏版本目录
- 寻找 `replay_recordings` 目录 
- 将对应的replay文件放入目录即可(要注意游戏版本)
